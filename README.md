# Rayford AI Public Website

This repository publishes the public Rayford AI website at
[rayford-ai.com](https://rayford-ai.com).

It intentionally contains only public-facing website files: static pages,
website assets, team images, research figures, and the Ray Assess demo used for
company communication.

Product code, customer discovery, strategy, model workflows, data pipelines, and
venture materials live in private Rayford AI repositories.

## Repository Policy

This repository is public for visibility and company communication, but it is
not open source. All rights are reserved unless Rayford AI grants written
permission.
